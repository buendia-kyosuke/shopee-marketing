## クイックサマリー

### 確定している戦略方針
1. **PH + SG並走**でスタート（TW は競合強すぎるため保留）
2. PHは**実験場・注文数確保**、SGは**利益の主柱**
3. ツールで同一商品をPH/SG同時出品、国別マージンを分けて設定
4. 6〜9ヶ月後にMYかTHへ拡張検討

### 重要な気づき
- 薬局系・お菓子・定番スキンケアは定価以下で赤字しか出ない
- 狙うべきは**グローバルニッチ**（日本にあるがPH/SGにない専門品）
- Xの「PH売上報告が多い」は初心者集中バイアスであり、稼げる国の指標ではない
- 実際の高収益セラーはSGが売上の60〜65%を占める
