## 日本人セラーの典型的な参入パターン

```
Step 1: Shopee PHでスタート（ほぼ全員）
Step 2: Preferred Seller取得（PH）
Step 3: SG追加

多くのセラーが Step 1〜2 で止まっている
→ XのPH投稿が多い理由
```

## 競合セラーの伸ばし方・分析手法

### 競合調査の手順

1. Shopeeで売れ筋カテゴリの上位セラーを特定
2. 確認項目:
   - 商品数（少ない数で高売上 = ニッチの証拠）
   - 更新頻度（毎日更新 = アクティブな競合）
   - レビュー内容（不満点 = 自分が改善できるポイント）
   - 送料設定・配送日数

### 差別化の具体的手法

- 日本語説明 + 英語 + 現地語の多言語対応
- バンドル販売（単品より20〜30%高く売れる）
- 日本国内でのストーリー訴求（「日本在住者が選んだ」）
- 使い方解説（専門知識が参入障壁になる）

## 重要な業界インサイト

### せどりの構造的問題

```
「売れ筋商品」の運命:
発見 → 出品 → 競合参入 → 価格競争 → 利益消滅

解決策:
→ 商品の回転速度を上げる（常に新商品発掘）
→ OR ニッチに逃げる（競合が来にくい場所）
→ OR バリューを加える（比較できないようにする）
```

### グローバルニッチの見つけ方

```
PHで「Is this available in Japan?」的なコメントがある商品
→ 需要確認済み・供給が追いついていない

Amazonレビュー100〜2,000件 × Shopee競合10件以下
→ 需要実証済みニッチの黄金条件
```
