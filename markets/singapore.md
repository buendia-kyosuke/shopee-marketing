# シンガポール（SG）市場

## 基本データ

| 項目 | 数値 |
|---|---|
| GDP/capita | 日本の約1.8倍 |
| EC市場における越境比率 | **60%**（EC全体の6割が越境購入） |
| 関税・GST免税枠 | SGD400以下は無税 |
| Shopee市場での立ち位置 | SG EC市場で最大訪問プラットフォーム |
| EC成長率予測 | +18%（2025年まで） |

## 現状評価

**確認済みの強み（実体験あり）**

- 高単価で売れることを確認済み
- PH比で1.5〜2倍の価格設定が可能
- 日本品質への信頼が厚い

## 実際のセラー収益事例

ミツ氏（月売上180万円セラー）の国別内訳:

```
SG: ¥1,183,000（64%）← 圧倒的メイン収益源
PH: ¥394,000（21%）
MY: ¥176,000（10%）
TH: ¥73,000（4%）
TW: ¥37,000（2%）
```

→ **SGが利益の柱になるモデルが実証されている**

## SGを優先すべき理由

1. **購買力**: 所得が高く、品質に対して適正価格を払う文化
2. **越境EC比率60%**: 海外から買うことに慣れている
3. **日本ブランド認知**: Made in Japanプレミアムが機能する
4. **シンプルな関税**: SGD400以下無税 = 一般的な商品は問題なし
5. **英語公用語**: 出品・CSが英語でOK

## SG Preferred Seller 取得戦略

```
目標: できるだけ早期取得（広告ツール・検索優遇解放）

条件（目安）:
- 30日間30注文以上
- ショップ評価4.5以上
- 非履行率5%以下

PHよりPreferred条件は若干厳しいが
1注文あたりの利益が高いので早期到達を目指す
```

## 禁止・規制カテゴリ

- **ペットフード**: 2026年3月時点で禁止カテゴリに追加済み

## SG向け商品戦略

### 強いカテゴリ

- 日本製スキンケア（SK-IIなどプレミアム帯）
- キッチン・調理器具（日本品質評価高い）
- 美容家電
- 専門道具・ホビー系

### 価格帯

- SGD10〜50（¥1,100〜5,500）が主戦場
- PHでは売りにくい高単価商品をSGで展開する

## 手数料（2025年時点）

- コミッション: 14%（2025年5月改定後）
- 注意: PHの5.6%と比べて高いが、販売価格が高いため絶対利益は大きい

## キャンペーン情報

- 11.11・12.12 は安定した高ボリューム
- SG Shopee公式は日本品の越境購入を積極的に推進
