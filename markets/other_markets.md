# その他市場分析

## 台湾（TW）

### 基本データ

| 項目 | 内容 |
|---|---|
| 日本文化親和性 | SEA最強（75%が日本をお気に入り国と回答） |
| Shopeeのポジション | TW EC市場で最大手アプリ |
| 日本セラー参入率 | 32.7%（日本人セラーが最も多く参入する国） |

### 評価: **現時点では参入しない**

**実体験による判断:**
- 日本に最強クラスのセラーがすでに確立済み
- 他のどの国よりも赤字になりやすい
- 「参入率32.7% = 競合が最も集中している」の裏返し

**重要な教訓:**
> 「日本人セラーが最も多く参入している国」≠「最も稼げる国」
> 人気 = 飽和 = 価格競争

### 将来的な参入条件

- 強豪セラーが手を出していないニッチカテゴリが特定できた場合のみ
- 現状は保留
