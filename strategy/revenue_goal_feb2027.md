## 戦略の3本柱

### 1. SGファースト（利益の67%）

SGが全体利益の2/3を担う。SG失敗 = 目標未達。

- **最優先**: SGニッチ商品のリサーチ・出品を毎週継続
- **Preferred取得**: 7月までに取得し広告ツール解放
- **単価UP**: バンドル・セット販売で平均注文額を引き上げ

### 2. ニッチポートフォリオ（持続的な利益源）

1カテゴリに依存しない分散構造を作る。

```
目標: 5カテゴリ以上で各月利 ¥50,000〜150,000

例:
- 日本製釣り具 → SG/MY
- 理美容プロ用品 → SG/PH
- キッチン専門器具 → SG/TH
- 園芸・盆栽用品 → SG/MY
- ホビー・模型用品 → SG/PH
```

### 3. セール最大化（ブースト収益）

11.11 / 12.12 / 2.2 の大型セールで平時の2〜3倍を狙う。

- セール前の在庫確保（2週間前に仕入れ完了）
- セール専用の広告キャンペーン
- セール限定バウチャーで新規顧客獲得
- セール後のリピーター追跡

## 正直な現実チェック

### これは楽観シナリオか？

**部分的にはYes。** 以下が前提条件:

1. **SGで安定的にニッチ商品が見つかり続ける** — 最大の不確実性
2. **月200注文@SG** — Preferred取得後の広告活用が前提
3. **2.2セールで+30万円ブースト** — 実績データなしの推定
4. **MY/THが合計月16万円** — 市場調査はこれから

### 保守的シナリオ（SG伸び悩み時）

```
SG 150注文 × ¥2,000 = ¥300,000（楽観比 -¥165,000）
→ 2月月利: ¥700,000〜750,000
```

### 100万円達成を確実にするために

- **毎週のSGリサーチを絶対にサボらない**（週10品の新規出品を継続）
- **データで判断**: 売れない商品は2週間で撤退、売れる商品に広告を集中
