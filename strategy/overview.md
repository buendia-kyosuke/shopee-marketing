# 全体戦略概要

## 目標

**1年以内に月利100万円達成**

現状：週3注文（月12注文）
必要成長率：利益1,000円/注文なら月1,000注文（83倍）

---

## 現状のビジネスモデルの構造的問題

### なぜ「わかりやすい人気商品」が詰んでいるか

```
薬局系・お菓子・定番スキンケア
→ 日本人セラー多数 + 中国セラー大量出品
→ 定価以下の価格競争が常態化
→ 利益300円/注文が「良い方」の水準
```

せどりの宿命:
```
売れる商品を見つける
→ 競合が同商品を安く出品
→ 価格競争で利益消滅
→ また新しい商品を探す（繰り返し）
```

**これはトレッドミル。商品選定だけでは解決しない。**

## 戦略の核心：グローバルニッチ

### 定義

「日本では普通に買えるが、対象国では入手困難かつ需要がある商品」

```
[Amazonで仕入れ可能]
      ∩
[対象国Shopeeで需要あるが供給薄]
      ∩
[中国からのコピー品が出にくい]
      ↓
グローバルニッチの黄金地帯
```

### 競合が生まれにくい理由

| 要因 | 説明 |
|---|---|
| 発見コスト | セラーが「その商品の存在」を知らない |
| 需要の見えにくさ | 対象国の需要がAmazonランキングに出ない |
| 説明コスト | 使い方説明が必要 → 参入障壁になる |
| 量が少ない | 中国大手が量産体制を組みにくい |

### 優先カテゴリ例

- 趣味・専門道具（日本製釣り具、特定スポーツ用品）
- プロ用品のBtoC展開（理美容ツール、調理器具）
- 機能特化型（防災グッズ、園芸用品）
- 地方限定・非流通品

## 販売速度ブースト戦略（回転率優先商材）

### 問題：ニッチ商品だけでは注文数が伸びない

```
現状の構造:
  ニッチ商品 → 競合少ない → 利益率は良い
             → しかし需要自体が少ない → 注文数が伸びない
             → 週3注文で停滞
```

ニッチのポートフォリオ化だけでは、各ニッチの回転が遅く、
日次の注文数が安定しない。これがPreferred維持やショップの成長を阻害する。

### 解決策：低価格・高回転商材をベースラインとして投入

**目的は利益ではなく「注文数のベースライン確保」**

```
目標: 1日5注文以上を定常化

高回転商材（利益ゼロ〜薄利）: 1日3〜4注文 → ベースライン
ニッチ商材（利益あり）       : 1日1〜2注文 → 利益の源泉
────────────────────────────────────────────
合計                          : 1日5注文    → ショップの成長エンジン
```

### なぜ利益ゼロでも成立するか

1. **国内配送料の分散**: デフォルト300円/注文で計上しているが、1日の発送をまとめれば1件あたりのコストは下がる。注文数が増えるほど配送料負担が薄まる
2. **Preferred Seller維持・昇格**: 注文数が直接的に評価基準
3. **ショップの露出向上**: 注文数・レビュー数が増えるとShopeeアルゴリズムで優遇
4. **リピーター獲得**: 低価格商品で初回接点を作り、ニッチ商品への導線にもなる

### 高回転商材の選定基準

| 条件 | 基準 |
|---|---|
| 販売価格 | PH: PHP100〜300 / SG: SGD3〜10 |
| 競合価格 | 競合と同等かやや安い（価格で負けない） |
| 回転頻度 | Shopee上で月50件以上売れている実績がある商品カテゴリ |
| 利益 | ゼロ〜¥100/注文で許容 |
| 仕入れの安定性 | Amazon常時在庫あり・価格変動少ない |
| 軽量 | SLS送料を最小化するため500g以下推奨 |

### 注意点

- 高回転商材はあくまで**ベースライン確保の手段**であり、利益の源泉ではない
- 利益はニッチ商材（特にSG）で稼ぐ構造は変わらない
- 赤字になる商材は投入しない（利益ゼロが下限）
- 回転商材でもレビュー・評価は丁寧に管理する

## ニッチのポートフォリオ化

1つのニッチでは月利100万円の天井が低い。

```
目標構成（12ヶ月後）:
ニッチA: 月50注文 × ¥2,000 = ¥100,000
ニッチB: 月60注文 × ¥1,800 = ¥108,000
ニッチC: 月40注文 × ¥2,500 = ¥100,000
ニッチD: 月50注文 × ¥2,000 = ¥100,000
...
PH + SG合算で月利¥1,000,000
```

各ニッチを「専門店」として育てる意識が重要。
