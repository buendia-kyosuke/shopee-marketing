## 現状のビジネスモデルの構造的問題

### なぜ「わかりやすい人気商品」が詰んでいるか

```
薬局系・お菓子・定番スキンケア
→ 日本人セラー多数 + 中国セラー大量出品
→ 定価以下の価格競争が常態化
→ 利益300円/注文が「良い方」の水準
```

せどりの宿命:
```
売れる商品を見つける
→ 競合が同商品を安く出品
→ 価格競争で利益消滅
→ また新しい商品を探す（繰り返し）
```

**これはトレッドミル。商品選定だけでは解決しない。**

## 戦略の核心：グローバルニッチ

### 定義

「日本では普通に買えるが、対象国では入手困難かつ需要がある商品」

```
[Amazonで仕入れ可能]
      ∩
[対象国Shopeeで需要あるが供給薄]
      ∩
[中国からのコピー品が出にくい]
      ↓
グローバルニッチの黄金地帯
```

### 競合が生まれにくい理由

| 要因 | 説明 |
|---|---|
| 発見コスト | セラーが「その商品の存在」を知らない |
| 需要の見えにくさ | 対象国の需要がAmazonランキングに出ない |
| 説明コスト | 使い方説明が必要 → 参入障壁になる |
| 量が少ない | 中国大手が量産体制を組みにくい |

### 優先カテゴリ例

- 趣味・専門道具（日本製釣り具、特定スポーツ用品）
- プロ用品のBtoC展開（理美容ツール、調理器具）
- 機能特化型（防災グッズ、園芸用品）
- 地方限定・非流通品

1つのニッチでは月利100万円の天井が低い。

```
目標構成（12ヶ月後）:
ニッチA: 月50注文 × ¥2,000 = ¥100,000
ニッチB: 月60注文 × ¥1,800 = ¥108,000
ニッチC: 月40注文 × ¥2,500 = ¥100,000
ニッチD: 月50注文 × ¥2,000 = ¥100,000
...
PH + SG合算で月利¥1,000,000
```
