# 売上(GMV) 分解フレームワーク

Shopee Japan公式説明資料より。**全戦略アクションをこの3要素のどれを動かすかで分類する**ためのレンズ。

---

## 基本式

```
売上(GMV) = アクセス数(Visit) × 転換率(CVR) × 客単価(AOV)
```

---

## 3要素の構成要素とドライバー

### ① アクセス数 (Visit) — 商品ページに流入させる
- マーケット数やSKU数UP
- モール内広告（Shopee Ads）
- SNSマーケティング
- ライブストリーミング
- 商品数UP

### ② 転換率 (CVR) — 流入を購入に変換する
- フォロワー数UP
- レビュー数UP
- セール（値引き）
- 配送のリードタイム短縮
- 商品への安心感（説明・画像・返品体制）

### ③ 客単価 (AOV) — 1注文あたりの金額を上げる
- まとめ買いUP（Bundle Deal / 数量割引）
- セット商品

---

## 優先度ルール

```
アクセス数 > 転換率 > 客単価
```

**理由**: セラー側で動かせる可変性とインパクトの順。
- アクセス数: SKU追加・広告出稿で短期に効く。最初に伸ばすべき
- 転換率: レビューや配送実績の蓄積が必要で時間がかかる
- 客単価: 商品設計依存で打ち手が限定的

---

## フェーズ別の重点

| フェーズ | 重点要素 | 理由 |
|---|---|---|
| Phase 1 (Month 1-3) | **アクセス数** | SKU 50件登録、PH/SGで露出確保が最優先 |
| Phase 2 (Month 4-6) | **アクセス数 + 転換率** | Shopee Ads開始、レビュー積み上げ、Preferred取得 |
| Phase 3 (Month 7-9) | **転換率** | 主力商品の差別化、画像・説明最適化 |
| Phase 4 (Month 10-12) | **客単価（SG）** | SGで高単価ニッチ商品にシフト、AOVで月利を底上げ |

---

## アンチパターン

- 「客単価を上げよう」と高額商品から始める → アクセス数がないので注文ゼロ
- 転換率改善（A/Bテスト等）に時間を使いすぎる → 母数が少ないので統計的に意味がない
- まずアクセス数を積み上げ、母数が出てから次の要素を最適化する
