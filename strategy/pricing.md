# 価格・マージン戦略

## ツールの利益計算ロジック（3段階）

### 1. 販売価格計算 `calculate_sales_price()`

仕入値 → 販売価格

```
原価ベース(現地通貨) = (仕入値 + 国内送料 + SLS送料) / 為替レート
販売価格 = 原価ベース × 手数料乗数 × (1 + 利益率)

手数料乗数 = 1 / (1 - fee_rate)
```

**具体例（SG向け）:**
```
仕入値: 1,000 JPY
国内送料: 300 JPY
SLS送料: 3 SGD (約300 JPY)
為替: 1 SGD = 100 JPY
手数料率: 14% → 手数料乗数 = 1.163
利益率: 17%

原価ベース = (1,000 + 300 + 300) / 100 = 16 SGD
販売価格 = 16 × 1.163 × 1.17 = 21.7 SGD → 21.70 SGD
```

---

### 2. 利益逆算 `calculate_profit()`

販売価格から利益率と利益額を逆算

```
原価ベース_現地 = (仕入値 + 国内送料 + SLS送料) / 為替
利益率 = (販売価格 / (原価ベース_現地 × 手数料乗数)) - 1
利益(JPY) = (販売価格 - 原価ベース_現地 × 手数料乗数) × 為替
```

---

### 3. 実注文利益 `calculate_order_profit()`

Shopeeの実際の入金額から実現益を計算

```
実現益 = セラー手取り(現地通貨) × 為替 - 仕入値 - 国内送料
```

## pricing_config.json の設定値

| パラメータ | 値 | 用途 |
|---|---|---|
| `packaging_weight_kg` | 0.1 kg | SLS送料計算に加算 |
| `domestic_shipping_jpy` | 300 円 | 日本国内配送料（全国共通） |
| `default_profit_rate` | 17% | デフォルト利益率（PH除く） |
| SG/TW等の `fee_rate` | 14% | 手数料乗数 = 1/(1-0.14) = 1.163 |
| PHの `fee_rate` | 23% | 手数料乗数 = 1/(1-0.23) = 1.299 |

> ⚠️ PHは手数料23%と高いため、`default_profit_rate` が **2%** に設定されている

## 国別マージン設定方針

| 国 | ツール設定利益率 | 手数料率 | 備考 |
|---|---|---|---|
| SG | 17%（デフォルト） | 14% | 高単価・購買力高い・主力 |
| TW | 17%（デフォルト） | 14% | 参入しない判断済み |
| PH | 2% | 23% | 手数料高く薄利・注文数確保が目的 |

## SGの価格優位性

同一商品の場合:
```
PH販売価格: ¥1,200 → 利益¥300
SG販売価格: ¥2,400 → 利益¥1,200（同じ商品で4倍の利益）
```

実際の高収益セラー事例:
- ミツ氏（月売上180万円）の内訳: SG 64%・PH 21%・その他15%
- SG単体で月売上約115万円を実現
